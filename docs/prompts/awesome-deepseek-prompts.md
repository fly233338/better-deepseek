# Awesome DeepSeek Prompts

Better DeepSeek 浏览器扩展官方提示词集合。点击"刷新"即可在扩展中获取最新提示词，也欢迎提交 PR 贡献新提示词。

## 效率

### 提问优化器

把模糊问题改写成更容易得到高质量回答的提示词。

```text
请帮我优化下面这个问题，让它更清晰、更具体、更适合交给 AI 回答。

要求：
1. 先指出原问题中缺失的信息或可能的歧义。
2. 给出一个可直接复制使用的优化版提示词。
3. 如果有必要，补充 2-3 个可选追问。

原问题：
{{问题}}
```

tags: 效率, 提问

---

### 总结提炼

把长文本压缩成可行动的重点和待办。

```text
请总结下面内容。

输出格式：
1. 3-5 条核心结论。
2. 重要事实/数据。
3. 可执行待办。
4. 仍不确定或需要追问的信息。

内容：
{{内容}}
```

tags: 效率, 总结

---

### 文本摘要

将长文本压缩为简洁摘要。

```text
请为以下内容生成摘要，控制在 {{字数}} 字以内：

{{文本}}
```

tags: 总结, 效率

---

## 写作

### 写作润色

在不改变原意的前提下，让文本更清楚、更自然。

```text
请润色下面的文本。

目标风格：{{目标风格}}
读者对象：{{读者对象}}

要求：
1. 保持原意，不凭空增加事实。
2. 修正语病、重复和不自然表达。
3. 输出润色后的版本，并简要说明主要修改点。

原文：
{{文本}}
```

tags: 写作, 效率

---

### 中英互译

翻译文本，并保留语气、格式和专有名词一致性。

```text
请将下面内容翻译为{{目标语言}}。

要求：
1. 保留原文格式、列表和代码块。
2. 专有名词先按通用译法处理，不确定时保留原文。
3. 语气保持{{语气}}。

原文：
{{原文}}
```

tags: 翻译, 写作

---

### 写作风格模仿

模仿指定风格的写作。

```text
请用 {{风格}} 的风格写一段关于 {{主题}} 的文字，字数约 {{字数}}。参考以下风格特征：

{{风格特征}}
```

tags: 写作, 创意

## 编程

### 代码审查

审查代码的正确性、可维护性、性能和安全风险。

````text
请以资深工程师的方式审查下面的代码。

背景/目标：{{背景}}

请按优先级输出：
1. 可能导致错误或回归的问题。
2. 安全、性能或边界条件风险。
3. 可维护性改进建议。
4. 建议补充的测试用例。

代码：
```
{{代码}}
```
````

tags: 编程, 审查

---

### 代码生成

根据需求描述生成代码实现。

```text
请用 {{语言}} 编写一个函数，实现以下功能：

{{需求描述}}

要求：
- 包含适当的错误处理
- 添加类型注解
- 代码清晰可读
```

tags: 编程, 生成

---

### SQL 生成

将自然语言转换为 SQL 查询。

```text
请将以下需求转换为 {{数据库类型}} 的 SQL 语句：

{{需求}}

表结构：
{{表结构}}
```

tags: 编程, SQL

### 系统提示词 — 编程助手

适用于 DeepSeek 的编程助手系统提示词。

```text
你是一个专业的 {{语言/领域}} 开发者。请遵循以下准则：

1. 代码优先：尽量输出可直接运行的完整代码
2. 解释在后：代码后附上关键逻辑的说明
3. 遵循 {{语言/领域}} 最佳实践

用户需求：{{需求}}
```

tags: 编程, 角色

## 学习

### 学习导师

用循序渐进的方式讲清一个概念，并带练习。

```text
请扮演一位耐心的学习导师，帮助我理解：{{概念}}。

我的水平：{{当前水平}}

请按这个结构回答：
1. 一句话解释。
2. 用生活类比解释。
3. 关键概念拆解。
4. 常见误区。
5. 给我 3 道由浅入深的小练习，并附参考答案。
```

tags: 学习, 解释

---

## 分析

### 深度分析

从背景、利弊、风险和方案多角度分析问题。

```text
请对下面问题做一次结构化深度分析。

问题：{{问题}}

请包含：
1. 背景和关键约束。
2. 主要利益相关方。
3. 支持与反对观点。
4. 潜在风险和反例。
5. 2-3 个可选方案及取舍。
6. 你的建议和判断依据。
```

tags: 分析, 决策

---

### 方案对比

比较多个选项并给出推荐决策。

```text
请帮我比较以下几个方案，并给出推荐。

目标：{{目标}}
候选方案：{{候选方案}}
约束条件：{{约束条件}}

请输出：
1. 对比维度。
2. 表格化比较。
3. 每个方案的主要风险。
4. 推荐方案和理由。
5. 如果信息不足，需要我补充什么。
```

tags: 决策, 分析

## 创意

### 头脑风暴

围绕一个主题生成多方向创意，并筛选可执行方案。

```text
请围绕"{{主题}}"进行头脑风暴。

要求：
1. 先给出 12 个不同方向的点子。
2. 每个点子用一句话说明价值。
3. 按"新颖度/可行性/成本"简单打分。
4. 最后推荐最值得尝试的 3 个方案。
```

tags: 创意, 方案

---

### 角色

生成可用于角色扮演或长期对话的角色设定。

```text
请帮我创建一个可用于 AI 角色扮演的角色卡。

角色方向：{{角色方向}}
互动风格：{{互动风格}}

请输出：
1. 角色名称。
2. 身份背景。
3. 性格特点和说话习惯。
4. 知识边界和行为禁忌。
5. 开场白。
6. 可直接作为系统提示词使用的完整角色设定。
```

tags: 角色, 创意

---

## 推理

### 逐步推理

使用思维链进行复杂推理。

```text
请一步一步地思考以下问题，展示你的推理过程：

{{问题}}

请按照以下格式输出：
1. 问题分析
2. 已知条件
3. 推理步骤
4. 最终答案
```

tags: 推理, 思考

## 办公

### 周报与汇报

把零散工作记录整理成清晰的汇报材料。

```text
请把下面的工作记录整理成{{汇报类型}}。

受众：{{受众}}

要求：
1. 突出成果和影响，而不是流水账。
2. 问题和风险要客观具体。
3. 下一步计划要可执行。
4. 语气专业、简洁。

工作记录：
{{工作记录}}
```

tags: 办公, 写作
